bal – a dnscoin client
----------------------

This is a simple command-line interface to `dnscoin <http://dnscoin.nz/>`_, written in Python.

To install:

::

 $ pip install bal

To use:

::

 $ bal 3Nxwenay9Z8Lc9JBiywExpnEFiLp6Afp8v
 140955.82050855
 $ bal -s 3Nxwenay9Z8Lc9JBiywExpnEFiLp6Afp8v
 14095582050855
 $ cat whales.txt | bal
